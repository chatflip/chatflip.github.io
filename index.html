Resource Lens

Monitor CPU, memory, and GPU usage directly in your VS Code status bar.

CPU, memory, and GPU usage displayed in the status bar. Hover for detailed tooltips.

Features

💻

CPU Monitoring

Real-time CPU usage displayed directly in the status bar at a glance.

🧠

Memory Tracking

Live memory consumption metrics so you always know your system load.

🎮

NVIDIA GPU Support

Monitor GPU utilization, VRAM, and temperature via nvidia-smi. Auto-hides if unavailable.

📊

Interactive Tooltips

Hover over any metric for detailed bar charts and additional statistics.

⏱️

Configurable Interval

Adjust the refresh rate from 500ms up to any interval you prefer.

🔧

Selective Display

Toggle individual metrics on or off. Show only what matters to you.

Settings

Setting Default Description
resourceLens.updateIntervalMs 1000 Update interval in milliseconds (min 500)
resourceLens.showCpu true Show CPU usage in the status bar
resourceLens.showMemory true Show memory usage in the status bar
resourceLens.showGpu true Show GPU usage in the status bar
